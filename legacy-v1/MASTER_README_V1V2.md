# SaaS App Master Workspace

This project workspace is organized into major versions representing the evolutionary stages of the SaaS platform.

## 📁 Repository Structure

- **[`/v1-animation-challenge`](./v1-animation-challenge)**: The original animation project featuring a single-file application structure with integrated Supabase logic. Includes utility scripts for secure refactoring.
- **[`/v2-studio-boilerplate`](./v2-studio-boilerplate)**: The next-generation SaaS boilerplate following a clean architecture with a feature-based modular structure.
  - Built with: Next.js 16+, TypeScript, Tailwind CSS, Prisma, and Zod.
  - Design Language: **Minimal Brutalism**.
