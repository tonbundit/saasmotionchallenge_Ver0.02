# SaaS Motion Challenge — 50 Weeks

This project is a 50-week challenge for SaaS animation/motion design. It includes a progress tracker and integration with Supabase for data persistence and authentication.

## Features
- **50-Week Progress Tracker**: Track your journey week by week.
- **Supabase Integration**:
  - **Google OAuth**: Secure login using Google.
  - **Data Sync**: Your progress is automatically saved to Supabase.
- **Interactive UI**: Modern, responsive design with glassmorphism and animations.

## Tech Stack
- HTML / Vanilla CSS / JavaScript
- [Supabase](https://supabase.com/) (Auth & Database)
- Google Fonts & Material Symbols

## Setup
1. **GitHub Login**: Use the Google Login button in the app.
2. **Database Sync**: Ensure the `user_progress` table is created in your Supabase project (refer to the SQL editor instructions).

## Hosting
Deployable on [Vercel](https://vercel.com/) or any static hosting service.
